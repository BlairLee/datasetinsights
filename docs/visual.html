visual.decode_segmap(labels, dataset='cityscapes')¶

Decode segmentation class labels into a color image.

Parameters

  • labels (np.array) – an array of size (H, W) with integer grayscale

  • denoting the class label at each spatial location. (values) –

  • dataset (str) – dataset name. Defaults to “cityscapes”.

Returns

A np.array of the resulting decoded color image in (H, W, C).

Note

(H, W, C) stands for the (height, width, channel) of the 2D image.
